# Naveen Sakthi B. | Portfolio Website

A premium, modern, and fully responsive personal portfolio website showcasing the skills, academic history, projects, and certifications of **Naveen Sakthi B**, an Electronics and Communication Engineering student specializing in IoT, Embedded Systems, and Automation.

---

## 🛠️ Tech Stack & Architecture

- **Structure**: Semantic HTML5 markup for structural hierarchy and SEO optimization.
- **Styling**: Vanilla CSS3 featuring:
  - **Dynamic Theme Mode**: High-tech circuit-inspired dark mode (default) with neon cyan/emerald accents and clean slate light mode.
  - **Custom Animations**: Slide-ins, hover scaling effects, and interactive grid card elevations.
  - **Layout**: CSS Flexbox and Grid layouts optimized for responsive breakpoints (Mobile, Tablet, Desktop).
- **Interactivity**: Vanilla JavaScript implementing:
  - **Local Storage Caching**: Persists theme mode selection across browser restarts.
  - **Typewriter Effect**: Loops key specializations dynamically in the hero header.
  - **Live Search & Filters**: Instant category filters for projects and query-based text search for certifications.
  - **Interactive Contact Form**: Custom form validation and status alerts.
- **Icons & Typography**: 
  - [Feather Icons](https://feathericons.com/) for modern vector elements.
  - *Plus Jakarta Sans* and *Fira Code* Google Fonts.

---

## ✨ Key Sections & Features

1. **Header & Navigation**: Clean floating navbar with glassmorphism backdrop blur and theme switching toggle.
2. **Hero Section**: Catchy greetings, dynamic typewriter animations, social connections, and profile photo.
3. **About & Education**: Profile summary and B.E. ECE details mapped on a visual timeline.
4. **Skills Badges**: Technical and programming skills grouped into responsive, glowing hover-state tags (removing typical percentage bars for a cleaner look).
5. **Krish Tech Experience**: Detailed interactive card listing embedded systems intern achievements.
6. **Project Grid**: Showcases *Home Automation*, *Robotic Arm*, and *Toll Booth Monitoring* with category filtering buttons.
7. **Searchable Certificates**: Filterable grid containing Autodesk, MathWorks, MongoDB, and NPTEL achievements.
8. **Contact Form**: Interactive validation and mock submission feedback.

## 🚀 How to Run Locally

### 1. Direct Access
You can open the website instantly by double-clicking the [index.html](index.html) file in your file explorer to open it in any web browser.

### 2. Local Development Server
To run with a local server (recommended for testing responsive features or caching):

**Using Node.js (`npx`):**
```bash
# Navigate to project directory
cd C:\Users\kavya\.gemini\antigravity\scratch\naveen-portfolio

# Start a lightweight server
npx http-server
```
*Open `http://localhost:8080` in your browser.*

**Using Python:**
```bash
python -m http.server 8000
```
*Open `http://localhost:8000` in your browser.*
